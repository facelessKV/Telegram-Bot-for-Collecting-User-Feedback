# ИНСТРУКЦИЯ ПО УСТАНОВКЕ И ЗАПУСКУ БОТА ДЛЯ СБОРА ОТЗЫВОВ

## ЧТО ДЕЛАЕТ БОТ

Этот бот собирает отзывы от пользователей в Telegram и сохраняет их в базу данных. Администратор может просматривать все отзывы.

## УСТАНОВКА НА WINDOWS

### Шаг 1: Установка Python 3.9

1. Скачайте Python 3.9.13 с официального сайта:
   https://www.python.org/downloads/release/python-3913/

2. Выберите "Windows installer (64-bit)" и скачайте файл.

3. Запустите установщик и обязательно отметьте галочку "Add Python 3.9 to PATH" внизу окна установщика.

4. Нажмите "Install Now".

5. Дождитесь завершения установки и нажмите "Close".

### Шаг 2: Скачивание файлов бота

1. Создайте на рабочем столе папку с названием "feedback_bot".

2. Скопируйте файл с кодом бота в эту папку и назовите его "bot.py".

### Шаг 3: Установка необходимых библиотек

1. Нажмите Win+R, введите "cmd" и нажмите Enter, чтобы открыть командную строку.

2. В командной строке перейдите в папку с ботом, набрав:
   ```
   cd C:\Users\ИМЯ_ПОЛЬЗОВАТЕЛЯ\Desktop\feedback_bot
   ```
   (Замените ИМЯ_ПОЛЬЗОВАТЕЛЯ на имя вашего пользователя в Windows)

3. Установите необходимую библиотеку, введя:
   ```
   pip install aiogram==3.0.0
   ```

## УСТАНОВКА НА LINUX

### Шаг 1: Установка Python 3.9

1. Откройте терминал с помощью Ctrl+Alt+T.

2. Обновите список пакетов:
   ```
   sudo apt update
   ```

3. Установите необходимые инструменты:
   ```
   sudo apt install software-properties-common
   ```

4. Добавьте репозиторий с Python:
   ```
   sudo add-apt-repository ppa:deadsnakes/ppa
   ```

5. Установите Python 3.9:
   ```
   sudo apt install python3.9 python3.9-venv python3-pip
   ```

### Шаг 2: Создание папки и файла бота

1. Создайте папку для бота:
   ```
   mkdir ~/feedback_bot
   ```

2. Перейдите в эту папку:
   ```
   cd ~/feedback_bot
   ```

3. Создайте файл bot.py с помощью текстового редактора (например, nano):
   ```
   nano bot.py
   ```

4. Вставьте код бота, нажмите Ctrl+O для сохранения, затем Enter и Ctrl+X для выхода.

### Шаг 3: Установка библиотеки

1. Установите aiogram:
   ```
   pip3 install aiogram==3.0.0
   ```

## НАСТРОЙКА БОТА

### Шаг 1: Получение токена бота

1. Откройте Telegram и найдите @BotFather.

2. Отправьте сообщение: /newbot

3. Следуйте инструкциям BotFather:
   - Укажите имя для бота (например, "Feedback Collector")
   - Укажите username для бота (должен заканчиваться на "bot", например, "my_feedback_bot")

4. BotFather пришлет вам токен, выглядящий примерно так: 1234567890:AABBCCDDeeffGGHHiijjKKLLmmNNooPPqq

5. Скопируйте этот токен.

### Шаг 2: Определение вашего Telegram ID (для админа)

1. Найдите в Telegram бота @userinfobot

2. Отправьте ему любое сообщение

3. Бот ответит вам сообщением, в котором будет указан ваш ID (например, "Your id: 123456789")

4. Запомните или запишите этот ID

### Шаг 3: Настройка файла бота

1. Откройте файл bot.py в любом текстовом редакторе.

2. Найдите строку:
   ```
   API_TOKEN = 'YOUR_BOT_TOKEN'
   ```
   
   Замените 'YOUR_BOT_TOKEN' на токен, который вы получили от BotFather (обязательно сохраните кавычки).

3. Найдите строку:
   ```
   ADMIN_ID = 123456789
   ```
   
   Замените 123456789 на ваш Telegram ID, который вы получили от @userinfobot (без кавычек).

4. Сохраните файл.

## ЗАПУСК БОТА

### В Windows:

1. Откройте командную строку (Win+R, введите "cmd" и нажмите Enter).

2. Перейдите в папку с ботом:
   ```
   cd C:\Users\ИМЯ_ПОЛЬЗОВАТЕЛЯ\Desktop\feedback_bot
   ```

3. Запустите бота:
   ```
   python bot.py
   ```

4. Бот запущен! Не закрывайте окно командной строки, пока хотите, чтобы бот работал.

### В Linux:

1. Откройте терминал (Ctrl+Alt+T).

2. Перейдите в папку с ботом:
   ```
   cd ~/feedback_bot
   ```

3. Запустите бота:
   ```
   python3.9 bot.py
   ```

4. Бот запущен! Не закрывайте терминал, пока хотите, чтобы бот работал.

## ИСПОЛЬЗОВАНИЕ БОТА

1. Откройте Telegram и найдите вашего бота по имени, которое вы указали при создании.

2. Отправьте команду `/start`, чтобы получить приветственное сообщение.

3. Чтобы оставить отзыв, отправьте команду `/feedback`, а затем напишите ваш отзыв.

4. Администратор (чей ID указан в настройках) может просмотреть все отзывы, отправив боту команду `/all_feedbacks`.

## ВАЖНЫЕ ЗАМЕЧАНИЯ

1. Бот будет работать только пока запущен скрипт в командной строке/терминале.

2. Если вы хотите, чтобы бот работал постоянно, вам потребуется хостинг или VPS.

3. Все отзывы сохраняются в файл базы данных feedback.db в той же папке, где находится файл bot.py.

## УСТРАНЕНИЕ ПРОБЛЕМ

1. Если при запуске возникает ошибка "ModuleNotFoundError: No module named 'aiogram'", проверьте, что вы правильно установили aiogram:
   ```
   pip install aiogram==3.0.0
   ```
   
2. Если бот не отвечает на команды, проверьте, что вы указали правильный токен.

3. Если вы не можете просмотреть отзывы как администратор, проверьте, что вы указали правильный ID администратора.

4. Если бот перестал отвечать на команды, перезапустите его (закройте командную строку/терминал и запустите снова).
